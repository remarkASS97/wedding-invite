# Публикация приглашения и подключение Google Таблицы

## 1. Google Таблица для ответов

1. Создайте Google Таблицу, например `Ответы свадьба Антон и Ирина`.
2. В таблице откройте `Расширения` → `Apps Script`.
3. Удалите стандартный код и вставьте содержимое файла `google-apps-script/Code.gs`.
4. Нажмите `Сохранить`.
5. Нажмите `Развернуть` → `Новое развертывание`.
6. Тип: `Веб-приложение`.
7. `Выполнять от имени`: `Меня`.
8. `У кого есть доступ`: `Все`.
9. Нажмите `Развернуть` и разрешите доступ.
10. Скопируйте URL веб-приложения. Он выглядит примерно так:

```text
https://script.google.com/macros/s/AKfycb.../exec
```

## 2. Вставить URL в приглашение

В файле `wedding-invite-assets/wedding-mini-site.html` найдите строку:

```js
const GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/PASTE_YOUR_DEPLOYMENT_ID_HERE/exec';
```

Замените её на URL вашего веб-приложения из Apps Script.

## 3. GitHub Pages

1. Создайте новый публичный репозиторий на GitHub, например `wedding-invite`.
2. Загрузите в него файлы из этой папки.
3. В репозитории откройте `Settings` → `Pages`.
4. В `Build and deployment` выберите:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Сохраните.

Через пару минут сайт будет доступен по ссылке:

```text
https://ВАШ_ЛОГИН.github.io/wedding-invite/
```

## 4. Проверка

Откройте опубликованную ссылку, заполните анкету тестовыми данными и проверьте, что в Google Таблице появилась новая строка.
