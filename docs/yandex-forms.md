# Подключение анкеты через Яндекс Формы

## 1. Создать форму

1. Откройте Яндекс Формы: `https://forms.yandex.ru/cloud/admin`
2. Нажмите `Создать форму`.
3. Назовите форму, например `Анкета гостей: свадьба Ирина и Антон`.
4. Добавьте вопросы:
   - `Ваше имя` — короткий текст, обязательный.
   - `Вы сможете быть с нами?` — один вариант, обязательный: `Да, с радостью буду`, `К сожалению, не смогу`.
   - `Предпочтения по напиткам` — один или несколько вариантов: `Вино`, `Игристое`, `Крепкие напитки`, `Безалкогольные напитки`, `Другое`.
   - `Пищевые ограничения` — длинный текст.
   - `Комментарий` — длинный текст.

## 2. Включить сбор ответов

1. Откройте настройки формы.
2. Проверьте, что форма опубликована и доступна по ссылке.
3. В разделе ответов включите уведомления на почту, если хотите получать каждый ответ письмом.
4. Скопируйте публичную ссылку формы. Она будет похожа на:

```text
https://forms.yandex.ru/u/XXXXXXXXXXXXXX/
```

## 3. Вставить ссылку в сайт

В файле `wedding-invite-assets/wedding-mini-site.html` найдите строку:

```js
const YANDEX_FORM_URL = 'https://forms.yandex.ru/u/PASTE_YANDEX_FORM_ID/';
```

Замените её на публичную ссылку вашей формы:

```js
const YANDEX_FORM_URL = 'https://forms.yandex.ru/u/XXXXXXXXXXXXXX/';
```

То же самое нужно сделать в `publish/wedding-invite-assets/wedding-mini-site.html`, если публикуете файлы из папки `publish`.

## 4. Проверить

Откройте сайт, нажмите `Заполнить анкету`, отправьте тестовый ответ и проверьте, что он появился в Яндекс Формах.

## 5. Вернуть гостя к приглашению после ответа

1. Откройте форму и перейдите в `Настройки` → `Тексты и логика отправки`.
2. В разделе `Событие после отправки` включите `Перенаправлять на сайт`.
3. Укажите адрес:

```text
https://remarkass97.github.io/wedding-invite/wedding-invite-assets/rsvp-complete.html
```

4. Нажмите `Сохранить` и опубликуйте изменения формы.

После отправки ответа встроенная форма свернётся. Если гость открыл анкету в отдельной вкладке, он автоматически вернётся к приглашению.
